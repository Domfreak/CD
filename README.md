## **Setup Instructions**

Follow these steps to get the bot running on your server:

### 1. Clone the Repository

```bash
git clone https://github.com/Domfreak/CD.git
cd CD
```

### 2. Install Dependencies

Make sure you have Python 3.8 or higher installed.

```bash
pip install -r requirements.txt
```

### 3. Create a `.env` File

Create a `.env` file in the root directory and add your Discord bot token:

```
DISCORD_TOKEN=your-bot-token-here
PREFIX=your-custom-prefix
```

### 4. Run the Bot

```bash
python bot.py
```

## **How to Use**

1. Invite the bot to your server using the OAuth2 URL from the Discord Developer Portal.
2. Use the custom prefix followed by `help` to view a list of all available commands.
3. Enjoy!
